Exam-gap rule
Between exams
Day 1: full-syllabus revision · Day 2: timed paper + correction · Day 3: weak points only, stop heavy work by 4pm
Biology
  • Draw every system completely from memory on a blank page before checking — neuron, nephron, sarcomere, alimentary canal, heart, lung
  • Cornell notes for each system: left = key term, right = function/mechanism, bottom = 3-line summary
  • Teach each system aloud as if explaining to someone who knows nothing
  • Flashcards for all hormones, receptors, and feedback loops — test yourself daily
  • Since both papers cover the full syllabus, use paper 1 feedback to target weak systems for paper 2
Chemistry
  • Solve a minimum of 10 numerical problems per chapter — stoichiometry, equilibrium, and electrochemistry need repeated drilling
  • Build one A4 organic reaction map linking alkanes → alkenes → alkynes → benzene → alcohols → esters — reproduce it from memory each revision day
  • Make a single-page formula and constant sheet; cover it and rewrite it from memory each morning
  • For conceptual chapters (thermodynamics, bonding) write short explanatory paragraphs — not bullet points
  • Both papers cover everything: use paper 1 to find which chapters need more problem practice before paper 2
Physics
  • Derive every formula from first principles rather than memorising — if you understand the derivation you can reconstruct it in the exam
  • Draw and fully label every circuit type: capacitor charge/discharge, RL, AC, transformer — do this from memory each session
  • Structured problem approach: identify the type → write all relevant equations → solve → check units
  • Materials Physics LO 12–14 is conceptual — focus on definitions, key phenomena, and distinguishing characteristics
  • Both papers are full syllabus: use paper 1 to find which topics need more derivation practice
Geology
  • Draw the full rock cycle and water cycle from memory — annotate every stage and process
  • Practice interpreting seismograms step-by-step: identify P and S wave arrivals → calculate distance → locate epicenter
  • Study geological and distribution maps of Egypt — know where coal, petroleum, seismic zones are located
  • For geophysics: make a table of method → what it measures → what property it detects → application
  • Remote sensing: learn the vocabulary and classification of techniques; practice describing false-color image processing steps
Arabic (one exam — full syllabus)
  • Nahw: write your own example sentences for every rule — reading the rule alone is not enough
  • Novel: make a character map and a chapter-by-chapter event timeline for all 22 chapters
  • Balagha: build a table of literary devices with one original example for each
  • Literature: write a short paragraph summary for each of the 15 lessons, then memorise key lines
  • Timed writing practice: one analytical paragraph in 20 minutes each day of Phase 2
English (one exam — 6 cycles)
  • 3 cycles per revision day — summarise vocabulary, grammar points, and key themes for each
  • Write one short paragraph per cycle using the target vocabulary so words stick in context
  • Read your summary notes aloud — hearing yourself boosts retention for language
  • Timed passage practice: read + answer comprehension questions within the time limit
German (one exam — 14 Lektionen)
  • 7 Lektionen per day — grammar structures first, vocabulary second, example sentences third
  • Write one short paragraph per Lektion using key words — production beats passive reading
  • Focus especially on verb conjugations, cases, and any irregular forms — these appear in every exam question
  • Read dialogues aloud to internalise sentence rhythm
