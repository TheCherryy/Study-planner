# Gamified Study Planner

Pink, playful exam planner with tabs for timeline, weekly, daily routine, methods, tracker, resources, and a game hub.

## Features

- Full study planner from April to June with weekly structure.
- Interactive tracker and resources checklists with progress bars.
- Character system with XP, levels, streak, badges, and confetti.
- Game Hub with:
  - Daily quests
  - Boss battles
  - Coins and reward shop
  - Skill tree
  - Focus timer
  - 30-day heatmap
  - Exam countdown + next exam banner
- Persistent progress with `localStorage`.

## Files

- `study_plan_corrected_apr21_jun18.html` - main app
- `favicon.svg` - cute site icon

## Run locally

Open `study_plan_corrected_apr21_jun18.html` in your browser.

## Change the icon

Replace `favicon.svg` with your own file and keep this line in the HTML:

```html
<link rel="icon" type="image/svg+xml" href="favicon.svg">
```
